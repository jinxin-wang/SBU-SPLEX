Genes UP - GO Molecular Function Annotations
 
Used notations:
 
  • List Hits - the number of genes annotated by the considered GO Molecular Function category or annotation cluster within the analyzed list of target genes
  • List Total - the number of genes within the analyzed list of target genes having at least one GO Molecular Function annotation
  • Population Hits - the number of genes, available on the entire microarray, annotated by the considered GO Molecular Function category or annotation cluster
  • Population Total - the number of genes available on the entire microarray and having at least one GO Molecular Function annotation
  • P-value - the significance p-value of the gene enrichment of the considered GO Molecular Function category or annotation cluster, calculated with a unilateral Fisher exact test
